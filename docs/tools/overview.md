# Tools Overview

TwinCat-MCP exposes **50+ MCP tools** across these areas: instance management,
tree navigation, POU management, libraries, build, solution management, tasks,
system management, and ADS runtime access — plus three always-available
licensing tools.

A full, categorized tool reference is being prepared. In the meantime, your MCP
client lists every available tool with its description once connected (see
[Agent Connection](../getting-started/agent-connection.md)).
